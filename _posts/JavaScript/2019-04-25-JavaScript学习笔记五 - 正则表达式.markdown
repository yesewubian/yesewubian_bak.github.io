## 正则表达式



### 创建

#### 字面量

```javascript
var my_regexp = /"(?:\\.|[^\\\"])*"/g;
```
需要注意的是用字面量的方式创建的对象共享同一个单例


#### 构造器RegExp

```javascript
var my_regexp = new RegExp("\"(?:\\\\.|[^\\\\\\\"])*\"",'g');
```

### 元素

一个正则表达式包含一个或多个正则表达式因子，每个因子都能选择是否跟随一个量词，这个量词决定着这个因子被允许出现的次数。如果没有指定这个量词，那么该因子只会被匹配一次。
